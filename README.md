# Browser Chess (Vanilla HTML/CSS/JS)

A polished single-page chess app for two human players on the same device, with full legal move enforcement.

## File structure

- `index.html` — semantic app layout and UI containers
- `style.css` — responsive styling, board visuals, and modal styling
- `script.js` — chess state, legal move generation, rules enforcement, and rendering

## Run locally

1. Download or clone this folder.
2. Open `index.html` directly in your browser.

No build step, no backend, and no dependencies are required.

## Deploy on GitHub Pages

1. Push this project to a GitHub repository.
2. In GitHub, open **Settings → Pages**.
3. Under **Build and deployment**, set:
   - **Source:** `Deploy from a branch`
   - **Branch:** `main` (or your default branch)
   - **Folder:** `/ (root)`
4. Save settings and wait for GitHub Pages to publish.
5. Open the generated GitHub Pages URL.
